LampLighter Case Study

This page contains the case study for LampLighter, a two-dimensionsonal survival horror game originally created for the 2023 Spooky 2d Jam hosted on itch.io.

Here, I'll go into more detail about the game and my development process.

Play Here

Project Overview

While the official entry to the jam was a mixed success, it was the most successful outcome I'd had from any game jam so far. Previously, my final products had been missing a large amount of crucial elements, fundamentally lacked engaging gameplay, or had various other issues. For this particular game jam I'd had the foresight to scope the game more appropriately, and had enough experience under my belt to manage the project and execute the implementation effectively enough that it had been mostly complete at the end of the jam.

It was still lacking in a few elements of polish that we simply didn't have time for at the very end of the 48 hours. Namely issues related to the character controller, animations, and the UI. However, unlike other jams, These issues were simple enough that I was able to spend the next week independently finishing them, and I reuploaded a full version of the game (version 1.0) on Halloween night, one week after the original jam. After this release, I'd receive further feedback from my friends, which allowed me to hone in on a few design issues with the game. While these issues took a bit more time to properly revisit, I continued fixing them and was able to release the final and current version of LampLighter (version 2.0) at the start of December.
